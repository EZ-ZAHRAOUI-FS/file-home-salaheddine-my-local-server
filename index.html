💡 Spin Up a Local Server

Got your HTML files ready? Run this in your terminal to serve any folder on http://localhost:9111:

python3 -m http.server 9111
  1. Open terminal or CMD/PowerShell.
  2. cd into your project folder:
    3. cd /path/to/your/folder
  3. Type python3 -m http.server 9111 and hit Enter.
  4. In your browser, head to http://localhost:9111.
  5. 🎉 Boom—you’re live locally!
